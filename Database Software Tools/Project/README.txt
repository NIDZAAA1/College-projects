Package Transport System


This project is part of the coursework for the "Database Software Tools" class at the University of Belgrade, School of Electrical Engineering for the academic year 2023/2024.

The aim is to develop a comprehensive system for package transportation, including functionalities for user management, package handling, and delivery logistics.
